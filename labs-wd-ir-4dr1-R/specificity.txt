2.
When multiple rules point to the same element, the browser uses a scoring system to decide who wins.

For color, the browser uses darkorange because p is the weakest, .info is stromger than p but weaker than the id tags, 
#warning is very strong but #warning.info is the strongest because it combines an id and class. that's why the browser applies the proprieties
from #warning.info block over all the other ones.

For font family, the browser uses 'Courier New', monospace because #warning.info doesn't define any font, so the browser applies the next font that
has the highest specificity, therefore it uses the font family from #warning

3.

The browser chooses to apply the color defined in the head tag of the html file over the one defined in the linked css sheet because
the attributes defined inside the head label of the document have a higher priority.
